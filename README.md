# pdfBookCollect
## 介绍
一些高质量电子书资源分享   
不可否认书籍尤其高质量书籍对于程序员的重要性，不管是学而时习之，还是温故而知新。   
书籍在当前社会是一种步伐相对落后的资料，读一本书之前一定要做好评估，要读就认真读多次读。   
ps:
> 所有书籍类资源都逃不过版权问题，这个层面上还是希望有能力的人支持正版   
> **书籍有版权，知识却没有，任何学习的行为都值得被鼓励**
